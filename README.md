# ppg-hr-estimation

## Overview

This project contains jupyter notebooks for data exploration, preprocessing, and heart rate estimation from ppg and 3-axis accelerometer data from the PPG-DaLiA dataset.

## Setup

```
python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python -m ipykernel install --user --name=ppg-hr-estimation --display-name "ppg-hr-estimation"
```
